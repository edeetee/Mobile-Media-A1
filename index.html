ArtLink

ArtLink is an interactive app that allows users to search an entire art database using their phone camera.

It has a very minimal interface, only a large camera view and a little text card down the bottom. It is made to be quick and intuitive so that it seems like magic for the user.

The app will run very fast as searching can be calculated on a server, and the UI will not be heavy or slow to run.

Using SIFT or another computer vision algorithm and an image database like Te Papa Collections, we can find a visual match. The live screen shows some basic information on the bottom of the screen, as well as the actual image matched in the database.

The user can tap or swipe up on the information at the bottom of the screen to get more information about this piece. Such as A Flask from Te Papa Collections

Here is the view from dragging or clicking on the last information panel. You can still see the live image feed in the background, but while in this window, the app will not poll for new images. This design consideration is important as users don't want the whole page to disappear halfway through reading.

The app will also have a feature for vision impaired. Users can enable a dictation mode where directing the camera around will dictate titles of art pieces. By clicking on the screen or shaking the device, it will give the listener more detailed information about a work.

ArtLink could be a hub for interactive art phone experiments too, one such idea includes having the video of the phone feed into a maching learning algorithm that ties semantic t-SNE positioning to a few dimensions in audio, such as chords, rhythm, progression; allowing a blind user to experience visual art through sound.
